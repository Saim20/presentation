# Process Design & Production

## Hybrid Production System

### Continuous Production
- High-demand standardized products
- Household items, pipes, furniture
- 24/7 operations with minimal changeovers

### Batch Production
- Specialized/seasonal products
- Custom industrial components
- Promotional items

## Manufacturing Technologies
- **Injection Molding:** Chairs, tables, kitchenware
- **Extrusion:** Pipes and profiles
- **Blow Molding:** Containers and bottles
- **7,500+ molds** for product variety
